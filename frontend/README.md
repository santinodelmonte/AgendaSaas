# AgendaSaaS Frontend

Frontend en React 18 + TypeScript + TailwindCSS + React Router + Axios.

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Variables de entorno opcionales

Puedes adaptar `src/api/axios.ts` para usar `VITE_API_URL` si tu backend vive en otro dominio.
